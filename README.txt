README.txt
==========

Le repo hello-world est utilisé pour un exercice pour "openclassroom" sur
l'usage de "git" et "git-hub".

Le contenu n'a pas d'importance particulière.
Le repo est utilisé pour manipuler la commande "git" et le site "git-hub".
Il contient un script Perl qui lit un fichier XML en provenance d'Excel.
L'exemple est cependant fonctionnel.

Pour les curieux souhaitant l'utiliser, se placer dans le répertoire origine et lancer

    perl read_nab.pl nab.xml

Le résultat est l'affichage du fichier XML sous la forme d'une structure Perl.

Pour le moment, les scripts de test sont non fonctionnels. La commande :

	proove ./t

renvoie donc une erreur. Ce sera pour la suite de l'aventure :-)
